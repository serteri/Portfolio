# Portfolio Web site

<br>

### This is a web site about Serter Iyigunlu who is a professional web designer. You can reach me throuh the web site and see my jobs.The purpose of this website is to express potential employer and show my experience and ability to create different approaches for creating websites,

 <br>
 I used mobile first approach. I developed the web site for mobile view first. Navigation bar has two main parts. First one is logo where located on the left side of the nav bar. Second part has Home , About me , Portfolio and Contact me links.

 <br>

In the main section of the Home page. I divided this part into two parts. Left side has My name (Serter Iyigunlu to introduce) and a link to see my Portfolio. Rigth side has a form to join our newsletter.

Secon part of the main section blog examples have been added. You can see five different blogs with titles and images.

Last part is the footer. In this part you can acces my github,my linkedin and twitter.

On the About-me page, there is one main part which includes four different stages. First one is my experience ,second one my education. In the third section you can download my resume. There is a link to see my blog page at the last part of the section.

On the Portfolio page, you can see my jobs as a web developer. I added four jobs with pictures.

On the contact-me page, there is a form that you can fill out. You need to first name,last name ,email address and message. There is a radio button to as you whether you want to join our newsletter or not.

You can acces blog home page via About-me page by clicking click below to see my blog part.

On the blog page, there are total five blog entries, with blog titles and images. You can access each blog by clicking either images or titles. When you click one of them,you go to indiviual blog page.

Each blog page has blog title, blog image and blog content.
